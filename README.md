```
Luo virtuaali ympäristö:
> python -m venv venv
Aktivoi virtuaaliympäristö:
Linux: >source /venv/bin/activate
Windows: > /venv/scripts/activate
Asenna riippuvuudet:
> pip install -r requirements.txt
Käynnistä sovellus:
> flask run

```
