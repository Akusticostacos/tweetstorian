#TweetStorian - Trending In History

What?

TweetStorian is a web application that fetches Twitter's daily trending data and saves it into a database

Why?

So we're able to go back in time, and see what was on people's mind on a certain day in history.

How?

TweetStorian gets the trending data through Twitter's API during peak tweeting hours to ensure the datas credibility
